# KARIVO X 2.0

Modernes Next.js/React-Projekt für KARIVO.

## Lokal starten
```bash
npm install
npm run dev
```

## Deployment
Am besten mit Vercel:

1. GitHub Repository erstellen
2. Diese entpackten Dateien hochladen
3. Vercel mit GitHub verbinden
4. Framework wird automatisch als Next.js erkannt
5. Deploy
