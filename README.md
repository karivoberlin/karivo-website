# KARIVO X 2.2

Modernes Next.js/React-Projekt für KARIVO.

## Neu in 2.2
- Premium Preloader
- Cursor Glow
- stärkere Hero-Motion
- animierter Lichtstrahl
- Sales-Band / Conversion-Sektion
- bessere Motion Delays
- mehr Agentur-Look

## Lokal starten
```bash
npm install
npm run dev
```

## Deployment
Am besten mit Vercel:

1. ZIP entpacken
2. Dateien in GitHub hochladen
3. Vercel mit GitHub verbinden
4. Framework: Next.js
5. Deploy
