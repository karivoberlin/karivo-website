# KARIVO X 2.1

Modernes Next.js/React-Projekt für KARIVO.

## Neu in 2.1
- stärkerer Hero
- Mouse Parallax im Hero
- bessere Portfolio-Mockups
- Prozess-Sektion
- hochwertigere Cards
- bessere Formular-Fokuszustände
- sauberere Komponentenstruktur

## Lokal starten
```bash
npm install
npm run dev
```

## Deployment
Am besten mit Vercel:

1. ZIP entpacken
2. Dateien in GitHub hochladen
3. Vercel mit GitHub verbinden
4. Framework: Next.js
5. Deploy
